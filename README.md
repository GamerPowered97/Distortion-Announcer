# Distortion Tracker

Distortion Tracker is a simple Android app to track active Destiny 2 Distortion zones. It displays the current active zone, what's coming up next, and sends push notifications to your device so you don't miss them while in orbit.

## Features
*   **Calibrate**: Select the active zone in-game to sync the app's tracker.
*   **Cycle Viewer**: Shows the current and next zone (Dreaming City, Savathûn's Throne World, Moon, Europa, Nessus, Cosmodrome, EDZ) as they rotate through the system.
*   **Alerts**: Set up 20-minute and 5-minute notification reminders. The text of these alerts changes to fit the situation, but we won't spoil the surprise here!

## Installation & Setup
1. Go to the [Releases](https://github.com/GamerPowered97/Distortion-Announcer/releases/latest) page and download `app-release.apk` on your phone.
2. Open the app and grant **Notification** and **Exact Alarm** permissions.
3. Turn off **Battery Optimization** when prompted so the background timer stays alive.
    *   *Note: If you are on a Samsung, Xiaomi, or OnePlus phone, you may also need to manually set the app's battery setting to 'Unrestricted' and turn on 'Auto-start' so the system doesn't kill it in the background.*

## Solar Cycle Order
The zones rotate every hour in this order:
$$\text{Dreaming City} \rightarrow \text{Savathûn's Throne World} \rightarrow \text{The Moon} \rightarrow \text{Europa} \rightarrow \text{Nessus} \rightarrow \text{Cosmodrome} \rightarrow \text{EDZ}$$
